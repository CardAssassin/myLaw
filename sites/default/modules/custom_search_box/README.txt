-- SUMMARY --

The Drupal Custom search box module allows administrators to change the display
of the search box by a user-friendly form located on the Search settings page. 
This is done using hook_form_alter() and hook_search().

For a full description of the module, visit the project page:
  http://drupal.org/project/custom_search_box

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/custom_search_box


-- REQUIREMENTS --

* Search module


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* Configure user permissions in Administer >> User management >> Permissions >>
  search module:

  - administer search

    Users in roles with the "administer search" permission will be able to see and change
    the Custom search box settings.

* Customize the Custom search box settings in Administer >> Site configuration >>
  Search settings.
